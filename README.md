# Cool Games & Insert

An indie collection of browser bookmarklets, lightweight tools, games, and experimental web projects.

## Projects

### Ad Skipper
A lightweight bookmarklet that automatically clicks YouTube's visible Skip Ad button locally in your browser.

### Browser Games
A collection of browser-playable games and experiments.

### Utilities
Small browser tools and experimental scripts.

---

## What Is A Bookmarklet?

A bookmarklet is a lightweight browser tool saved as a bookmark.

It runs locally in the current webpage when clicked.

---

## Safety & Privacy

- No executable downloads
- No installation required
- Runs locally in your browser
- Can be removed anytime by deleting the bookmark
- No account login required
- No personal data collection
